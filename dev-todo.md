## Brief set status

**Locked at 31 epics (A through AE) on 2026-06-05.** Every brief carries a `**Pack:**` header per the catalogue in `PRODUCT.md` §5a. The platform is a **modular product** — `core` is the always-on foundation (Sales + Lettings), every other pack is an opt-in. Subsequent additions to the brief set require an amendment PR against this index and a corresponding change to `audit/audit-report.md`.
