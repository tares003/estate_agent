Property grid skeleton

Same dimensions as the real property card so content snaps in without shifting. The image holds --ratio-card.

A polite live region announces "Loading properties…" to screen readers while the skeleton shows.

Admin table skeleton

Rows mirror the column layout. On mobile this becomes stacked-card skeletons (per the responsive table rule).

Spinner, progress & inline

For full-page transitions, determinate jobs (SAR export, bulk import), and in-context loads. Progress bars use linear easing.
