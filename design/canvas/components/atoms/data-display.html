Status badge — market status

Maps 1:1 to market_status via --colour-status-*. Colour is never the only signal: each badge has an aria-label and a visible label.

Skeleton — loading placeholder

Pulses opacity 0.4 → 0.8 over 1500 ms (motion-spec §4). Under reduced-motion it holds a static dimmed state.
